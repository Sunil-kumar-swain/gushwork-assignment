# 🌐 Gushwork Frontend Assignment

This project is a responsive web page built using **vanilla HTML, CSS, and JavaScript**, following the design and requirements provided in the assignment.

---

## 📌 Features Implemented

### ✅ 1. Pixel-Perfect Design
- Implemented UI based on provided Figma design
- Maintained spacing, typography, and layout consistency

### ✅ 2. Fully Responsive
- Desktop, Tablet, and Mobile compatible
- Used Flexbox & CSS Grid for layout

### ✅ 3. Sticky Header
- Appears after scrolling beyond first fold
- Smooth transition animation
- Hides when scrolling up

### ✅ 4. Image Carousel with Zoom
- Interactive carousel
- Hover zoom effect on images
- Smooth transitions and animations

### ✅ 5. Clean Code & Best Practices
- Semantic HTML5 structure
- Modular CSS
- Optimized JavaScript
- Cross-browser compatibility

## 🛠️ Tech Stack
